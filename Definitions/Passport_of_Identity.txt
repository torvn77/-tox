Паспорт идентичности (заверяется основной цифровой подписью)
	Публичная часть
		1) ID паспорта
		2) Ник(Одна идентичность может создавать разные паспорта Петя, Мария и т.п.)
		3) ID "Идентичности"(опционально)
		4) Удостовреряющая цифровая подпись "Идентичности" при указании ID "Идентичности"
		   или самоподписной сертификат при её отсутствии.
		5) Номер версии паспорта(опционально, при указании ID "Идентичности")
		6) Дату создания паспорта(опционально)
		7) Область действия паспорта в виде пар из ID и именён чатов(опционально)
		8) Публичный ключ для подписи создаваемых чатов.
		9) Публичный ключ для подписи отправляемых сообщений.
	         Разделять все три подписи надо для минимизации последствий компрометации одной из подписей.
    	     Цифровые подписи для чатов и сообщений, а так же сам паспорт объявляются 
    	     не действительными или скомпрометированными специальным сообщением подписанным 
    	     одной из подписей паспорта или подписи идентичности
    	10) Комментарии к паспорту(в них к примеру можно указать интересы, цели и 
    	    способы связи с владельцем паспорта)(опционально)
	Приватная часть дополнительно содержит приватные ключи цифровых подписей.
