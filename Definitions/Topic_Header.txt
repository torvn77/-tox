Заголовок чата/темы (открыт, не шифруется)
	1)  Название чата
	2)  ID чата
	3)  Список тёгов
	4)  Краткое описание чата для показа в поисковых таблицах, ограниченное по числу символов
	5)  Полное описание чата в формате bb коде или Медиавики.
	6)  Ключ для шифрования корневых постов веток
	7)   Всё это подписыватся цифровой подписью из паспорта для создания чатов/тем или при его отсутствии самоподписным сертификатом
	8)  Флаг не указания даты создания веток.(постящий может перекрыть запрет)
	9)  Флаг не указания даты создания постов.(постящий может перекрыть запрет)
	10) Флаг не указания ID.(постящий может перекрыть запрет)
	11) Флаг не указания ников.(постящий может перекрыть запрет)
	12) Полная копия публичной части паспорта создателя(опционально).
